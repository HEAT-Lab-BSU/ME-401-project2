    Returns: Q_reaction [W]

    # ASSUMPTION: heat exchanger recovery excluded here; subtracted in Module 5

    Inputs:  Q_reaction [W], Q_preheat [W]
    Returns: Q_net [W]
    Equation: Q_net = Q_reaction - Q_preheat
    Note: structured so heat exchanger recovery term can be added here in Module 5
